# Приложение для анализа графов
Приложение предоставляет графический интерфейс для создания, визуализации и анализа графов. Поддерживаются базовые алгоритмы теории графов, интерактивное редактирование, а также сохранение и загрузка графов в формате JSON.

## Основной алгоритм
- Поиск сообществ (алгоритм Core Expansion);

## Классические алгоритмы
- Поиск мостов с использованием DFS;

- Алгоритм Крускала для минимального остовного дерева.

## Чтение/сохранение графов
- Сохранение графа в JSON;

- Загрузка графа из JSON.

# Установка и запуск
- Клонируйте репозиторий:

```git clone https://github.com/username/graph-analyzer.git```

- Перейдите в директорию проекта:

```cd graph-analyzer```

- Соберите проект:

```./gradlew build```

- Запустите приложение:

```./gradlew run```
